适配手机：
1，图片适配(不同手机(不同的像素密度)，会去加载不同资源文件下的图片)
	（如果当前像素密度对应文件夹下，没有图片的时候，加载的高分辨率的图）
	dp能够适配吗？不能适配
	设计师在切图的时候一般会切两倍和三倍两套图，既xhdpi和xxhdpi

2，dimens.xml适配(不同像素密度的手机，加载不同的dimens文件下的值)
   设置了个性化分辨率手机的属性后，需要给一个相同的默认属性

3，layout布局适配(不同手机(不同的像素密度)，会去加载不同资源文件下的布局文件)
	layout-1280x720

4，java代码适配(获取屏幕的宽高，控件的宽高按屏幕的比例去做设置)
	当前的宽高需要优先设置在控件的父布局上，让后再作用在子控件身上


5，权重适配(剩余控件的一个分配的规则,线性布局中才可以使用权重，height = 0dp )

6，百分比适配：使用安卓百分比布局库 Android-percent-support
